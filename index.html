We can see that the button is an anchor ("a") element, without a href attribute.
Without the attribute, the anchor can't receive focus, as the browser doesn't consider this an interactive element.
A screenreader user would also have to deal with this issue, so how can we fix that?
Its really pretty trivial.
We can add a placeholder href attribute to an anchor.
And we can make a span focusable by adding the tabindex="0"
